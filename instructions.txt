1. $ git clone https://github.com/HasanCse/symfony-docker-master.git

2. $ cd symfony-docker-master

3. $ docker-compose up

4. Browse to https://localhost/validator
   It will show output of validated input.csv file

6. From command line
     php .\bin\phpunit .\tests\
    to run test cases.

7. To see code : main Validator controller is in symfony-docker-master\src\Controller

8. Input.csv file is in symfony-docker-master\src\Controller folder

9. To see test case: symfony-docker-master\tests
